## 📂 Repository Structure

The repository is logically partitioned to support both foundational learning and advanced competitive programming.

### 🐍 1. Python Core Mastery
Focuses on the building blocks of the Python language:
- **Basics & Control Flow:** Variables, operators, loops, and conditional statements.
- **Data Structures:** Deep dives into Lists, Tuples, Sets, and Dictionaries.
- **Functional Programming:** Custom functions, lambda expressions, and decorators.
- **Strings:** Intensive slicing, formatting, and manipulation exercises.

### 💡 2. Data Structures & Algorithms (DSA)
Optimized solutions for complex algorithmic challenges:
- **Arrays & Hashing:** Duplicate detection, Two Sum, Union of arrays.
- **Sorting & Searching:** Binary Search, Bubble Sort, Merge without extra space.
- **Greedy & Optimization:** Kadane’s Algorithm, Minimize the heights, Minimum Jumps.
- **Linear Data Structures:** Linked List node management, Stack, and Queue implementations.
- **Mathematics & Recursion:** Pattern printing, factorial logic, and backtracking.

## ▶️ How to Run

**Prerequisites:** Python 3.8+

```bash

git clone https://github.com/Aryan-Rathi/Python-Core-and-DSA-Problems---Aryan-Rathi.git
cd Python-Core-and-DSA-Problems---Aryan-Rathi

# Run any solution directly
python "Two Sum/two_sum.py"
```

No external libraries are required — all solutions use the Python standard library only.
